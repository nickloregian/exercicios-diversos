O que o JavaScript pode fazer

O JavaScript pode modificar elementos HTML. Como podem ser feitas essas modificações?

    1. Pode realizar alterações no conteúdo HTML
    2. Pode realizar alterações nos atributos do elemento HTML
    3. Pode realizar alterações nos estilos do elemento HTML
    4. Pode esconder ou mostrar o elemento HTML
    5. Pode criar ou excluir o elemento HTML
